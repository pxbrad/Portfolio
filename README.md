## Project Goal

The goal of this project is to **learn TailwindCSS** while building a **portfolio website** that will later showcase other projects I create.

This site will act as a **foundation portfolio**, where future work and experiments can be displayed.

## Development Command

Run Tailwind in watch mode:

```bash
npx tailwindcss -i ./input.css -o ./output.css --watch
```

This ensures Tailwind recompiles automatically when changes are made
